# UI Design Tokens

## Farben
- Primary: `emerald-700` fuer primaere Aktionen und aktive Navigation
- Background: `stone-50`
- Surface: `white` mit `border-stone-200`
- Text: `stone-900`
- Muted: `stone-600`

## Statusfarben
- Pending: `amber-500` / Badge `bg-amber-100 text-amber-700`
- Approved: `emerald-600` / Badge `bg-emerald-100 text-emerald-700`
- Rejected: `rose-600` / Badge `bg-rose-100 text-rose-700`
- Info: `sky-500` / Badge `bg-sky-100 text-sky-700`
- Neutral: `stone-500` / Badge `bg-stone-100 text-stone-700`

## Abstaende und Radien
- Haupt-Container: `max-w-6xl`, `px-4 sm:px-5 lg:px-8`
- Section-Abstaende: `py-2` bis `py-8`, je nach Inhaltsdichte
- Cards: `rounded-2xl`, `shadow-sm`
- Buttons: `rounded-xl`, `min-h-[44px]`

## Komponenten
- `Button`: `primary`, `secondary`, `ghost`
- `Card`: Grundflaeche fuer alle Panels
- `SectionCard`: Abschnitt mit Header, optionaler Aktion und Body
- `Badge`: zentrale Status- und Hinweis-Chips
- `PageHeader`: Seitentitel mit optionaler Aktionszeile
- `EmptyState`: einheitliche Leerseiten fuer Listen
